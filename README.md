# 功能
实现虎牙直播平台的弹幕语音播报和礼物致谢
# 致谢
基于[huya-danmu](https://github.com/BacooTang/huya-danmu "With a Title")开发.
# 依赖环境
- anaconda3(python3.7)
- [node](https://nodejs.org/zh-cn/download/ "With a Title")
# 相关库安装
```
pip install pyttsx3
```

```
npm install huya-danmu --save
```

# 快速部署
- 修改app.js中roomid
- 启动服务
```
node app.js
```
- 关闭服务<br>
 关闭终端或者ctrl+d
